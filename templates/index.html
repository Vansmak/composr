Connection Types

  • Agent: http://hostname:5005 (Composr Agent)
  • TCP: tcp://hostname:2375 (unsecured)
  • TLS: tcp://hostname:2376 (with certificates)
  • SSH: ssh://user@hostname
  • Local Socket: unix:///var/run/docker.sock

Note: Multi-host management is still in development. The Agent connection type is recommended for production use as it's more secure than exposing Docker directly.

Important: Limited or no support is available for connection types other than the Composr Agent. For best results and future compatibility, use the Agent connection method.
